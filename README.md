# 🏨 Royal Semimas Hotel Website

A modern and responsive hotel website built for **Royal Semimas Hotel** located in Bandundu.


## ✨ Features

* Responsive design (mobile, tablet, desktop)
* Room showcase with images and pricing
* Lightbox gallery for room details
* Smooth navigation
* WhatsApp booking integration
* Swiper.js sliders

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Swiper.js

## 📁 Project Structure

```
/css
/images
/fonts
index.html
index.js
```

## 🚀 Getting Started

1. Clone the repository

```
git clone https://github.com/your-username/royal-semimas-hotel-website.git
```

2. Open `index.html` in your browser
